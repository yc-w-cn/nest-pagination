# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-27

### 新增

- 初始版本发布
- 实现 `PaginationDto` 分页请求数据传输对象
- 实现 `PaginatedResponseDto` 分页响应数据传输对象
- 实现 `ApiPaginatedResponse` Swagger 文档装饰器
- 添加 `DEFAULT_PAGINATED_RESPONSE` 默认分页响应常量
- 支持页码、每页条数、排序和过滤功能
- 完整的 TypeScript 类型支持
- 与 Nest.js 和 Swagger 完全兼容
